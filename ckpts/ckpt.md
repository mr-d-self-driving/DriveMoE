# Data Directory

This directory should contain the PaliGemma weight.

## Required Downloads

Download the [PaliGemma pre-trained weights](https://huggingface.co/google/paligemma-3b-pt-224) and place it in this directory

## Expected Directory Structure

After downloading all components, the directory structure should look like this:
```
├── ckpts
│   └── paligemma-3b-pt-224
```
